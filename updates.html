Confusing Adventure

2/19/2017

So far the project has been a confusing adventure. This week specifically we ran into a lot of issues when we tried to change code in existing projects. We would use sample projects provided by Google and see that they worked with our registered device. The applications would also work when we changed the app id to the id that we got from registering the application. Whenever we tried to change the code in the receiver app the system would no longer work. There is no explanation for this and we are researching possible issues.

We contacted our customer and reached out to Google's cast developer support in hopes that we can figure out this issue. As of now we are at a major roadblock in the project and need to figure out a way to get this past this. The issues could be from registration, github integration issues, or even a hardware issue. At this point we cannot rule anything out. We plan to meet again within the week to troubleshoot more.

Eventful Week

2/12/2017

This first week was very eventful. While there was not much progress made in the actual project, the team was able to get a plan for what they need to do in order to get a satisfied customer.

After meeting with the customer, it was decided that the team would first focus on giving an existing game called Centipede Chromecast support. If that was successfully completed, then the team would focus on making their own game that has Chromecast support. Getting the Chromecast to work with a game is the main goal, whether that be our own game or the preexisting one. The team and the customer are getting along well, and the project will be successfully completed if our plan is followed well.

The team had a primary meeting on Friday to work on the website and get adjusted to using GitHub as the main communication source. Roxanne taught us how we can use GitHub and what we should do to help with the project. We also discussed future meetings and what we plan to get done at the next meeting. Overall, the team is getting along very well and is appearing to be a very functional team.
