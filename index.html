Netcast Project Page

a Chromecast implementation of the classic game Centipede

Using Chromecast, we are implementing a browser game that allows the user to use any Google Chrome compatible device as a controller and their TV as a display that communicates with the controller. Our main goals are as follows:

  • Create a Custom Receiver that can manipulate what is displayed when webpage is cast.
  • Adapt existing browser game, Centipede to be cast and controlled through the Chromecast.
  • Update the graphics and user experience of the game to give it a more modern feel.

For more information head on over to our Project page!

Developer Requirements:

There are some things that you have to get situated before you can run this application.

  1. Create a developer account at: https://cast.google.com/publish/#/overview
  2. Make sure all hardware is in developer mode and up to date.
  3. Register your chromecast device with the developer page above
  4. Register the game with the developer page by following the instructions on the page
  5. Once all is registered, you will be given an appID, copy that ID and paste it into sender.html
  6. If all is setup correctly, the game should connect and work properly.
